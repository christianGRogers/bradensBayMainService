Documentation for Bradensbay Website Builder

How to Use

Simply ask the AI to perform tasks related to website creation or management on your Ubuntu VM with public access. Login and type anything in the prompt. For instance, try: “Create a portfolio website for [Your Name] with my LinkedIn profile at [Your LinkedIn Link].”

The LLM will keep a 15 input context window but to reset this type "clear".

For custom domain DNS forwarding, please contact christian@bradensbay.com.

VM-Context

When you create an account with Bradensbay, you are provided with an Ubuntu VM instance. All AI prompts are processed by the Gemini API, giving the AI controlled access to your VM with certain guardrails. You can SSH into your VM to make changes or set up your own code. You may also pre-install libraries and inform the AI, for instance: "Do this knowing that XYZ software is already installed."

If you encounter connectivity issues, you might need to use the command:

export http_proxy=http://10.0.0.11:3128;

How to Connect

You’ll need PuTTY and PuTTYgen (or similar SSH tools) installed. Always download from a secure source to avoid eavesdropping.

  1. Generate an RSA 2048-bit key in PuTTYgen. Move your mouse over the blank area to create a random seed. Save your private key securely and copy your public key.
  2. Log in to Bradensbay and use the command addkey to add your public key.
  3. In PuTTY, set the host as bradensbay.com and specify the port displayed in your console.
  4. Navigate to SSH > Auth > Credentials, and select your private key.
  5. Connect, enter your username, and trust the connection if prompted.

Note: The user password generated for your VM is only for sudo privileges.
